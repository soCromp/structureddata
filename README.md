# Structured Data Challenge Benchmark



### Environment 
```bash
conda create --name sd python==3.12
pip install pandas 
conda install -c pytorch -c conda-forge ctgan
```

### Steps to run
```bash
chmod +x acquisition.sh
./acquisition.sh
python harness.py
```
